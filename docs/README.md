# Документация

## Назначение
Этот каталог содержит подробную документацию по проекту.

## Состав
- `docs/backend.md` - backend: архитектура, конфигурация, воркеры, пайплайн видео, тесты, профилирование.
- `docs/frontend.md` - frontend: структура, стили, API-клиент, сборка и запуск.
- `docs/api.md` - список endpoint-ов и примеры запросов.
- `docs/architecture.md` - схема сервисов и потоков данных.
- `docs/deploy.md` - деплой в production и чеклист выката.

## Связанные документы
- `README.md` - общий обзор проекта и быстрый старт.
